AdaAllegro5
===========

A thin binding to the Allegro 5 open source, cross-platform
game programming libary.

http://alleg.sourceforge.net/

The Allegro 5.0 reference manual can be found here:

http://alleg.sourceforge.net/a5docs/5.0.10/
